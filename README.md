# NestHub

NestHub is a premium, Hyderabad-focused real estate listing website built using HTML, CSS, and JavaScript.  
It is designed as an internship-ready project with a polished UI, property cards, search filters, a modal gallery, and city-specific locality browsing.

## Features

- Responsive homepage with marquee banner.
- Premium hero section with Hyderabad branding.
- Property listings with images, badges, and pricing.
- Search and filter by locality and property type.
- Quick-view modal with gallery.
- Save/favorite button for listings.
- Popular Hyderabad neighborhoods section.
- Testimonials section.
- Contact form.
- Mobile-friendly navigation.

## How to Run Locally

1. Download or clone the project folder.
2. Open `index.html` in your browser.
3. Navigate through the pages using the menu.
4. Use `properties.html` to test search and filters.
